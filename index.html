Features

    Severity Map

    A tool that uses a severity and scale index to show how serious the illness is along with the areas of high concern. It helps doctors quickly assess diseases in X-rays, making diagnoses easier and more accurate. This tool is based on GradCam.

    Multiple Models

    Multiple models are used to detect different illnessess and different organs ensuring high accuracy.

    Linguistic Features

    Includes translation to regional languages, highlighting important words and a tts feature to ensure maximum usage and inclusivity.
